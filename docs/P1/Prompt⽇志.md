# Prompt日志

## 与AI的交互记录

### 1. 访谈结果格式统一
**目的**：将各个成员的访谈结果以统一的格式整合起来，方便进行后续的需求分析  
**提示词**：请你帮我将文档中的各个访谈的结果的格式进行统一，不用修改内容，而是更改格式  
**结果**：AI生成的文档基本满足要求，格式统一且保留了原始内容，仅需少量调整

### 2. 用例图绘制工具学习
**目的**：学习如何使用专业的工具来绘制用例图  
**提示词**：我现在需要为一个软件项目绘制一个用例图，请你推荐一些专业的用例图绘制工具，并介绍一下如何使用这些工具。  
**结果**：AI向我推荐PlantText来绘制并且提供了使用教程，实操下来基本没有问题，效果也不错。

### 3. 需求规格说明书生成
**目的**：按照P1任务要求，让AI生成需求规格说明书  
**提示词**：请为一个校园互助服务平台生成完整的需求规格说明书，平台支持学生发布和接单各类互助需求（快递代取、学习辅导、二手交易、组队匹配等），目标用户为大学生。请按IEEE 830标准格式输出。  
**结果**：AI按照要求生成了需求规格说明书，结构完整但部分内容需要根据实际调研结果进行调整和补充。
